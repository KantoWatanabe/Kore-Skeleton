# Kore-Skeleton
A skeleton for creating applications with [Kore](https://github.com/KantoWatanabe/Kore).

## Installation
```bash
composer create-project korephp/kore-skeleton YOUR_APP_NAME
cd YOUR_APP_NAME
cp app/config/config-sample.php app/config/config.php
```

## Example

### Web App
```bash
composer start
```

### CLI App
```bash
bin/console example
```
